# tripplex

Package `tripplex` implements arithmetic for tri-perplex numbers. There are five types:

* `Int64`
* `Float64`
* `Int`
* `Float`
* `Rat`

Each tri-perplex value is printed in the form "(a+bs+cT+dsT+eU+fsU+gTU+hsTU)". This is similar to `complex128` values.
