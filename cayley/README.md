# cayley

Package `cayley` implements arithmetic for Cayley octonions. There are five types:

* `Int64`
* `Float64`
* `Int`
* `Float`
* `Rat`

Each Cayley octonion value is printed in the form "(a+bi+cj+dk+em+fn+gp+hq)". This is similar to `complex128` values.
