# hamilton

Package `hamilton` implements arithmetic for Hamilton quaternions. There are five types:

* `Int64`
* `Float64`
* `Int`
* `Float`
* `Rat`

Each quaternion value is printed in the form "(a+bi+cj+dk)". This is similar to `complex128` values.
