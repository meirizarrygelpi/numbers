# nplex

Package `nplex` implements arithmetic for nilplex numbers. There are five types:

* `Int64`
* `Float64`
* `Int`
* `Float`
* `Rat`

Each nilplex value is printed in the form "(a+bÎ±)". This is similar to `complex128` values. The nilplex numbers are more commonly known as dual numbers.
