# bipplex

Package `bipplex` implements arithmetic for bi-perplex numbers. There are five types:

* `Int64`
* `Float64`
* `Int`
* `Float`
* `Rat`

Each bi-perplex value is printed in the form "(a+bs+cT+dsT)". This is similar to `complex128` values.
