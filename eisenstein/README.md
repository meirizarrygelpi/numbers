# eisenstein

Package `eisenstein` implements arithmetic for Eisenstein numbers. There are five types:

* `Int64`
* `Float64`
* `Int`
* `Float`
* `Rat`

Each Eisenstein value is printed in the form "(a+bÏ‰)". This is similar to `complex128` values.
