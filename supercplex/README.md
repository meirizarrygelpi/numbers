# supercplex

Package `supercplex` implements arithmetic for super-complex numbers. There are five types:

* `Int64`
* `Float64`
* `Int`
* `Float`
* `Rat`

Each super-complex number value is printed in the form "(a+bi+cβ+dγ)". This is similar to `complex128` values.
