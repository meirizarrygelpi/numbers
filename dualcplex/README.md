# dualcplex

Package `dualcplex` implements arithmetic for dual-complex numbers. There are five types:

* `Int64`
* `Float64`
* `Int`
* `Float`
* `Rat`

Each dual-complex value is printed in the form "(a+bi+cΓ+diΓ)". This is similar to `complex128` values.
