# trinplex

Package `trinplex` implements arithmetic for tri-nilplex numbers. There are five types:

* `Int64`
* `Float64`
* `Int`
* `Float`
* `Rat`

Each tri-nilplex value is printed in the form "(a+bα+cΓ+dαΓ+eΛ+fαΛ+gΓΛ+hαΓΛ)". This is similar to `complex128` values.
