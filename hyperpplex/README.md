# hyperpplex

Package `hyperpplex` implements arithmetic for hyper-perplex numbers. There are five types:

* `Int64`
* `Float64`
* `Int`
* `Float`
* `Rat`

Each hyper-perplex value is printed in the form "(a+bs+cΓ+dsΓ+eΛ+fsΛ+gΓΛ+hsΓΛ)". This is similar to `complex128` values.
